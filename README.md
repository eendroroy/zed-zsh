## Installation

##### [antigen](https://github.com/zsh-users/antigen):

    antigen bundle eendroroy/zed-zsh

##### [zgen](https://github.com/tarjoilija/zgen):

    zgen load eendroroy/zed-zsh

##### [zplug](https://github.com/zplug/zplug):

    zplug "eendroroy/zed-zsh"
